# StarBound — Portafolio

Landing estático (HTML/CSS puro, sin build step) con el portafolio de servicios y productos de StarBound.

## Desplegar en Vercel

1. En Vercel: **Add New > Project > Import Git Repository**.
2. Selecciona el repo `CLIENTES`.
3. En **Root Directory**, selecciona `starbound-portfolio`.
4. Framework Preset: **Other** (sitio estático, no necesita build command).
5. Deploy.

Vercel detecta `index.html` automáticamente y lo sirve como página estática.
